        Vim Mode

        Full modal editing via @codemirror/vim. Toggle with CtrlAltV or :set vim / :set novim.

        Modes

        Esc / Ctrl-[ → Normal mode
        i Insert before cursor
        I Insert at line start
        a Insert after cursor
        A Insert at line end
        o New line below, insert
        O New line above, insert
        v Visual (char) mode
        V Visual line mode
        Ctrl-v Visual block mode
        R Replace mode

        Motion (Normal)

        h j k l ← ↓ ↑ →
        w / W Next word / WORD
        b / B Prev word / WORD
        e / E End of word / WORD
        0 / ^ Line start / first non-blank
        $ Line end
        gg / G File start / end
        :N / NG Go to line N
        { / } Prev / next blank line
        % Jump to matching bracket
        Ctrl-d / Ctrl-uHalf-page down / up
        Ctrl-f / Ctrl-bPage down / up
        f{c} / F{c} Find char forward / back
        t{c} / T{c} Till char forward / back
        ; / , Repeat f/t forward / back

        Editing (Normal)

        x / X Delete char under / before
        dd Delete line
        D Delete to end of line
        cc / C Change line / to end
        cw / ciw Change word / inner word
        ci" / ci' Change inside quotes
        ci( / ci[ Change inside parens/brackets
        yy / Y Yank (copy) line
        yw / yiw Yank word / inner word
        p / P Paste after / before
        u / Ctrl-r Undo / redo
        . Repeat last change
        J Join line below
        ~ Toggle case
        >> / << Indent / dedent line

        Search

        /pattern Search forward
        ?pattern Search backward
        n / N Next / prev match
        * Search word under cursor
        # Search word backward

        Counts & Operators

        3dd Delete 3 lines
        5j Move down 5 lines
        2yy Yank 2 lines
        d3w Delete 3 words

        Marks & Registers

        ma Set mark a
        `a Jump to mark a
        "ay / "ap Yank / paste to register a

        Ex Commands ( : )

        :w Save file
        :wq / :x Save and close
        :q! Discard changes
        :N Go to line N
        :%s/foo/bar/g Replace all in file
        :N,Ms/foo/bar/gReplace in lines N–M
        :Nd Delete line N
        :set number Show line numbers
        :set nonumber Hide line numbers
        :set novim Disable vim mode
        :noh Clear search highlight
